## Overiew

1. It is automation project using Selenium on ESPN Website.
2. Collecting data from website and Storing in Excel for analysing.
3. Writing unit test for testing the automation.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Prerequisite

* Python
* Selenium
* Excel
* Software testing Concepts
